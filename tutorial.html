Como instalar e configurar o Xampp e o PHP

O que é XAMPP:

O O XAMPP é um pacote de software livre e de códigos abertos que simplifica coniguração de um servidor web local.

O XAMPP é um acrônimo(é uma palavra formada pelas letras iniciais ou por partes de outras palavra).

Passo 1: Baixar o XAMPP

Visite esse site para baixar ele apachefriends.org e baixe a versão mais recente para o seu sistema operacional.

Passo 2: Instalar o XAMPP

  1. Após o download, execute o arquivo de instalação.
  2. Siga as instruções do assistente de instalação.
  3. Escolha os componentes que deseja instalar (geralmente, Apache, MySQL/MariaDB, PHP, e phpMyAdmin).
  4. Selecione o diretório de instalação e clique em "Instalar".
  5. Aguarde até que a instalação seja concluída.

Passo 3: Iniciar os serviços

  1. Após a instalação, abra o painel de controle do XAMPP.
  2. Inicie os serviços Apache e MySQL/MariaDB clicando nos botões "Start".





Passo 4:Verificar a Instalação

  1. Abra o seu navegador da web.
  2. Na barra de endereços, digite http://localhost e pressione Enter.
  3. Se o XAMPP estiver instalado corretamente, você verá a página inicial do XAMPP.

Pronto! Agora você instalou o XAMPP com sucesso e está pronto para começar a usar o PHP e outros recursos.
